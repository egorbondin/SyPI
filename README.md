## Install all dependencies (Установка всех зависимостей проекта)


### For Windows:
```
python -m venv venv

venv\scripts\activate.ps1

pip install -r requirements.txt
```

### For UNIX:
```
python3 -m venv venv

source venv/bin/activate

pip3 install -r requirements.txt
```
